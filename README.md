# Shop_plants_App

A new Flutter project.
An Plant Shop App.

Tech Stack :

 1 - > databases :  MongoDb .  
 
 2 - > rest Api : Java Spring boot.
 
 3 - > Ui : Flutter. 
 
 4 - > StateManagment : Provider .
